# Campus24by7 - Smart Hub

A modern, responsive web application designed to provide comprehensive campus management and student engagement solutions. Campus24by7 is a 24/7 intelligent hub for students and institutions.

## Project Overview

Campus24by7 Smart Hub is a web-based platform that streamlines campus operations and enhances student experience through integrated features, real-time information, and seamless communication tools.

## Technologies Used

This project is built with:

- **Vite** - Next generation frontend build tool for fast development
- **TypeScript** - Type-safe JavaScript for robust code
- **React** - UI library for building interactive components
- **Tailwind CSS** - Utility-first CSS framework for styling
- **shadcn-ui** - High-quality component library built on Radix UI

## Getting Started

### Prerequisites

- Node.js (v16 or higher) - [Install Node.js](https://nodejs.org/)
- npm or bun package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Pragya-josh/campus24by7-smart-hub.git
cd campus24by7
```

2. Install dependencies:
```bash
npm install
# or if using bun
bun install
```

3. Start the development server:
```bash
npm run dev
# or
bun run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
# or
bun run build
```
